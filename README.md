#Henry_CLI_project 

to create a to-do list, you can check is your things are done or not

##Description

The project is about you need to create a program about a to-do list. You can input and store a reminder, you can remove your reminder after you finished it. You can add date to remind yourself and you don't need to rerun the program while you add your new reminder.

##How it run

First, you enter your reminder and your date
Second, you can choose to add new reminders, if you stop add then it will give your reminder a number
last, if you finish your reminder, you can remove the reminder by entering the name
